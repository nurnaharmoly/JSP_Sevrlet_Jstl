Chapter 1 JSP Examples

The Basic date example shows how to use a scriptlet to print the current data using a java.util.Data object.

The Content.jsp uses the include directive to include a standard header and footer.

The MoreContent.jsp uses the same include directives, demonstrating how easy it is to include standard headers and footers on every JSP in your application.

The Date example using a scriptlet again shows using a scriptlet to print the current date. The next examples will show how to provide similar functionality using JavaBeans.

The Date example using a JavaBean with scriptlet provides the current data as a property of a JavaBean.

The Date example using jsp:getProperty uses the same JavaBean as the previous example, but accesses the current date property using the getProperty standard action.

The Date example using jsp:setProperty uses the setProperty standard action to configure the date format, and then uses getProperty to access the current date property.
